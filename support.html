Technical Support

Charging

Charging the Console

  • Charge your console by connecting an AC adapter directly to the console, or by using a Nintendo Switch adjustable charging stand (sold separately).
  • When you start charging, the battery icon ( ) at the top-right of the HOME Menu screen will change to ( ) to indicate that the console is charging.
  • When your console has reached 100% charge, the battery icon will change to ( ).
  • A complete charge takes approximately 3 hours when the console is in sleep mode

For more detailed help with charging issues, please check our troubleshooting section.

Internet Connection

You can connect to the Internet either wirelessly or via a LAN cable.


Setting up a wireless Internet connection:

Required components


  1. Place your Nintendo Switch console approximately 2 to 3 metres from your wireless router. Placing the console too far from the router may prevent it from establishing a connection.
  2. Press the HOME Button to display the HOME Menu, then select "System Settings"
  3. Select "Internet" → "Internet Settings".
  4. Under "Networks Found", select the SSID of the wireless router you're using.
  5. Enter the password for the wireless router and when "Connected" appears, the process is complete.

Your Nintendo Switch will remember this network and connect automatically when in range.

Setting up a wired Internet connection:

Required components


  1. Connect the USB hub to your Nintendo Switch Lite system.
  2. Connect a wired LAN adapter (sold separately) to a port on the USB hub, then use a LAN cable to connect the wired LAN adapter to your router.
  3. From this point, you should carry out the remaining instructions on your Nintendo Switch Lite. Press the HOME Button to display the HOME Menu, then select "System Settings".
  4. Select "Internet" → "Internet Settings".
  5. Select "Wired Connection" under "Registered Networks".
  6. Select "Connect to the Internet via Wired Connection". When "Connected" appears, the process is complete.

When setting up a wired connection, we recommend you also carry out the wireless setup.

With a wired connection, if you disconnect your console from your USB hub, the Internet connection will be lost. You're therefore advised to set up wireless Internet as well, so your console can switch over to a wireless connection whenever the hub is disconnected.

Users

Managing users on your Nintendo Switch:

  • Add a new user: Go to System Settings > Users > Add User
  • Link a Nintendo Account: Go to System Settings > Users > [select user] > Link Nintendo Account
  • Create child accounts: Add a supervised account through the Nintendo Parental Controls app
  • Transfer user data: Have both Switch consoles, go to System Settings > Users > Transfer Your User Data
  • Delete a user: Go to System Settings > Users > [select user] > Delete User

Each Nintendo Switch can have up to 8 user profiles. Linking a Nintendo Account enables online features and eShop purchases.

Friends

Managing your friends list:

  • Add a friend: Select your profile icon on the HOME Menu > Add Friend
  • Friend Code: Your unique 12-digit code can be found in your User Profile
  • Add friend methods:
    • Search by Friend Code
    • Search for Local Users
    • Search for Users You've Played With
    • Friends from other Nintendo apps (if linked)
  • Friend requests: Accept or decline in your profile page under Friend Requests
  • Remove a friend: Go to your profile, select the friend, and choose Remove Friend

You can have up to 300 friends on your Nintendo Switch. A Nintendo Switch Online subscription is required for many online features.

Common Issues & Solutions

Nintendo Switch Won't Charge

If your Nintendo Switch Lite isn't charging properly, try these troubleshooting steps:

  1. Use the original Nintendo Switch Lite AC adapter
  2. Check for any damage to the charging port or cable
  3. Let the console drain completely, then charge for at least 30 minutes before turning on
  4. Try a hard reset: Hold the Power button for 12 seconds, release, then press Power once
  5. Try charging directly to the console rather than through the dock

Wireless Connection Problems

If you're having trouble connecting your Nintendo Switch Lite to the internet:

  1. Move closer to your Wi-Fi router
  2. Restart both your router and Nintendo Switch
  3. Check if your Wi-Fi password has changed
  4. Try using a 2.4GHz network instead of 5GHz
  5. Update your system software if possible
  6. Manually set DNS settings (Primary: 8.8.8.8, Secondary: 8.8.4.4)
Watch video guide →

Game Card Not Reading

If your Nintendo Switch Lite isn't reading game cards:

  1. Remove and reinsert the game card
  2. Check for dirt or damage on the game card contacts
  3. Gently clean the game card slot with compressed air
  4. Restart your Nintendo Switch Lite
  5. Try another game card to see if the issue persists
  6. Update your system software
Watch video guide →

Excessive Battery Drain

If your Nintendo Switch Lite battery is draining too quickly:

  1. Lower screen brightness from Quick Settings (hold Home button)
  2. Turn on Airplane Mode when not using wireless features
  3. Close software completely when not in use
  4. Turn off controller vibration in System Settings
  5. Update your system software
  6. Consider replacing the battery if your console is older (requires service)
